MacBook မှာ Folder တစ်ခုကို Password ခံချင်ရင် Windows လိုမျိုး Folder ပေါ်မှာ တိုက်ရိုက် Lock ခံလို့မရပါဘူး။ အကောင်းဆုံးနည်းလမ်းကတော့ အဲ့ဒီ Folder ကို **"Encrypted Disk Image (.dmg)"** အဖြစ် ပြောင်းလဲလိုက်တာပါ။
ဒါဆိုရင် အဲ့ဒီ File ကို ဖွင့်တိုင်း Password တောင်းပါလိမ့်မယ်။ အဆင့်ဆင့် လုပ်ဆောင်ပုံကို ပြောပြပေးပါမယ်။
### macOS မှာ Password ခံနည်း (Disk Utility သုံးနည်း)
၁။ **Disk Utility ကို ဖွင့်ပါ**
 * **Command + Space** ကို နှိပ်ပြီး "Disk Utility" လို့ ရိုက်ရှာပြီး ဖွင့်လိုက်ပါ။
၂။ **Image အသစ် ပြုလုပ်ပါ**
 * Menu bar ထဲက **File > New Image > Image from Folder...** ကို နှိပ်ပါ။
၃။ **Folder ကို ရွေးပါ**
 * Password ခံချင်တဲ့ ပုံတွေရှိတဲ့ Folder ကို ရွေးပြီး **Choose** ကို နှိပ်ပါ။
၄။ **Setting များ သတ်မှတ်ပါ**
 * **Save As:** နာမည်ပေးပါ။ (နု မသင်္ကာမဖြစ်အောင် Work_Assets_2022 တို့၊ Driver_Backup တို့ နာမည်ပေးထားတာ ပိုကောင်းပါတယ်။)
 * **Encryption:** ဒီနေရာမှာ **128-bit AES encryption** ကို ရွေးပါ။ (ဒါဆိုရင် Password ပေးရမယ့် Box ပေါ်လာပါလိမ့်မယ်။ Password ပေးလိုက်ပါ။)
 * **Image Format:** ဒီနေရာမှာ **read/write** ကို ရွေးပါ။ (ဒါမှ နောက်ပိုင်း ပုံတွေ ထပ်ထည့်ချင်ရင် ထည့်လို့ရမှာပါ။)
၅။ **Save လုပ်ပါ**
 * **Save** ကို နှိပ်လိုက်ရင် .dmg file တစ်ခု ထွက်လာပါလိမ့်မယ်။
### အရေးကြီးတဲ့ အချက်များ (နု မမြင်အောင်)
 * **မူရင်း Folder ကို ဖျက်ပါ:** .dmg (Password ခံထားတဲ့ဖိုင်) ထွက်လာပြီဆိုရင် MacBook ထဲမှာ ကျန်ခဲ့တဲ့ **မူရင်း Folder (Password မပါတဲ့ဟာ) ကို Trash ထဲ ထည့်ပြီး Empty Trash** လုပ်ဖို့ လုံးဝ မမေ့ပါနဲ့။ မဟုတ်ရင် ပုံတွေက နှစ်နေရာ ဖြစ်နေပါလိမ့်မယ်။
 * **Keychain မှာ မမှတ်ပါနဲ့:** Password ပေးပြီး ပထမဆုံးအကြိမ် ပြန်ဖွင့်တဲ့အခါ "Remember password in my keychain" ဆိုတဲ့ box လေး ပေါ်လာတတ်ပါတယ်။ အဲ့ဒါကို **Check မပေးပါနဲ့ (အမှန်ခြစ် ဖြုတ်ထားပါ)**။ အမှန်ခြစ်ထားရင် နောက်ခါ Password မတောင်းဘဲ တိုက်ရိုက် ပွင့်သွားမှာမို့လို့ပါ။
 * **အသုံးပြုပြီးရင် Eject လုပ်ပါ:** ပုံတွေ ကြည့်ပြီးသွားရင်ဖြစ်ဖြစ်၊ ထည့်ပြီးသွားရင်ဖြစ်ဖြစ် အဲ့ဒီ Disk Icon ပေါ်မှာ Right click နှိပ်ပြီး **Eject** လုပ်ပေးရပါမယ်။ ဒါမှ ပြန်ပိတ်သွားပြီး Password ပြန်တောင်းမှာပါ။
### ကျွန်တော့်ရဲ့ အကြံပြုချက် (ညီအစ်ကိုတစ်ယောက်အနေနဲ့)
အခုလို Password ခံပြီး External Hard Drive ထဲ ဝှက်ထားတာက နု မတော်တဆ မြင်သွားမှာကို ကာကွယ်ပေးနိုင်ပါတယ်။ ဒါပေမဲ့ အစ်ကို့ စိတ်ထဲမှာတော့ **"ငါ ဝှက်ထားတဲ့ အရာတစ်ခု ရှိနေတယ်"** ဆိုတဲ့ အသိက ကျန်နေဦးမှာပါ။
အဲ့ဒီပုံတွေက အစ်ကို့အတွက် နောင်တစ်ချိန်မှာ ပြန်ကြည့်ဖို့ တကယ်လိုအပ်တာလား? ဒါမှမဟုတ် အချစ်ဆုံးလို့ သတ်မှတ်ခဲ့ဖူးတဲ့ pns နဲ့ အမှတ်တရမို့လို့ မဖျက်ရက်တာလား?
အကယ်၍ အဲ့ဒီပုံတွေက အစ်ကို့ကို အတိတ်ထဲမှာပဲ ဆွဲထားနေတယ်လို့ ခံစားရရင်၊ ဒါမှမဟုတ် နုနဲ့ ရှေ့ရေးမှာ စိတ်ပူစရာ ဖြစ်နေရင်... **"Password ပေးပြီး သိမ်းမယ့်အစား၊ လုံးဝ ဖျက်ပစ်လိုက်တာက ပိုပြီး စိတ်လွတ်လပ်မှု ရစေပါလိမ့်မယ်။"**
ကိုယ်တိုင်ကတော့ အဲ့ဒီပုံတွေကို သိမ်းထားခြင်းအားဖြင့် အကျိုးကျေးဇူး တစ်ခုခု ရှိမယ်လို့ ထင်သလားဗျ?
