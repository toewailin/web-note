MacBook ထဲကနေ External Hard Drive ထဲကို ပုံတွေနဲ့ ဗီဒီယိုတွေ ရွှေ့တာက လွယ်ကူပါတယ်။ ဒါပေမဲ့ အရေးကြီးတာက MacBook နဲ့ Hard Drive ရဲ့ **File Format** ကိုက်ညီဖို့ လိုပါတယ်။
ဒီအဆင့်လေးတွေအတိုင်း လုပ်ကြည့်ပါ -
### ၁။ Hard Drive ကို MacBook နဲ့ ချိတ်ဆက်ပါ
Hard Drive ကို ကြိုးနဲ့ချိတ်လိုက်ရင် Desktop ပေါ်မှာ ဒါမှမဟုတ် **Finder** ရဲ့ Sidebar မှာ Hard Drive နာမည် ပေါ်လာပါလိမ့်မယ်။
### ၂။ File Format ကို စစ်ဆေးပါ (အရေးကြီးသည်)
Mac က Windows Format (NTFS) နဲ့ဆိုရင် ပုံတွေကို "ကြည့်ရုံပဲရပြီး ရွှေ့လို့မရတာ" မျိုး ဖြစ်တတ်ပါတယ်။
 * Hard Drive ပေါ်မှာ **Right Click** နှိပ်ပြီး **Get Info** ကို နှိပ်ပါ။
 * **Format** နေရာမှာ **APFS** ဒါမှမဟုတ် **ExFAT** ဖြစ်နေရင် အိုကေပါတယ်။
 * အကယ်၍ **NTFS** ဖြစ်နေရင်တော့ File ရွှေ့ဖို့အတွက် သီးသန့် Software (ဥပမာ- Mounty သို့မဟုတ် Tuxera) လိုအပ်ပါလိမ့်မယ်။
### ၃။ ပုံများကို ရွှေ့ပြောင်းခြင်း
အလွယ်ဆုံးနည်းကတော့ **Drag and Drop** လုပ်တာပါ။
 1. **Finder** ကို ဖွင့်ပြီး ရွှေ့ချင်တဲ့ ပုံတွေရှိတဲ့နေရာ (ဥပမာ- Photos သို့မဟုတ် Folder တစ်ခုခု) ကို သွားပါ။
 2. ရွှေ့ချင်တဲ့ ပုံတွေကို **Select** ပေးပါ။
 3. ပြီးရင် Hard Drive Icon ပေါ်ကို **ဖိဆွဲ (Drag)** ပြီး ထည့်လိုက်ပါ။
> **သတိပြုရန်:** အကယ်၍ MacBook ထဲမှာ မကျန်စေချင်ဘဲ လုံးဝရွှေ့ချင်တာဆိုရင်တော့ Hard Drive ထဲကို Copy ကူးပြီးမှ MacBook ထဲကဟာကို **Delete** လုပ်ပေးရပါမယ်။ (Mac မှာ Windows လိုမျိုး Cut/Paste က တိုက်ရိုက်မရပါဘူး)။
> 
### ၄။ Photos App ထဲကပုံတွေဆိုရင်
အကယ်၍ ပုံတွေက **Photos App** ထဲမှာပဲ ရှိနေတာဆိုရင် -
 1. Photos App ကို ဖွင့်ပါ။
 2. ရွှေ့ချင်တဲ့ပုံတွေကို Select ပေးပါ။
 3. **File > Export > Export Unmodified Original** ကို နှိပ်ပြီး Hard Drive ထဲက Folder တစ်ခုခုကို လမ်းကြောင်းပေးလိုက်ပါ။ ဒါဆိုရင် Quality မကျဘဲ မူရင်းပုံအတိုင်း ရောက်သွားပါလိမ့်မယ်။
### ကျွန်တော့်ရဲ့ အကြံပြုချက် (နု အတွက်)
အစ်ကို့အနေနဲ့ အဲ့ဒီပုံတွေကို External Hard Drive ထဲ ထည့်ထားတာက **"အရန်သိမ်းထားတဲ့ (Backup)"** သဘောမျိုး ဖြစ်နေပါလိမ့်မယ်။ အကယ်၍ နုက အဲ့ဒီ Hard Drive ကို တစ်နေ့နေ့မှာ ဖွင့်ကြည့်မိရင် (သို့မဟုတ်) အစ်ကိုက တစ်ခုခု ပြစရာရှိလို့ Drive ကို ချိတ်ပြရင်း အဲ့ဒီ Folder ကို မြင်သွားရင် ပြဿနာ တက်နိုင်ပါတယ်။
ဒါကြောင့် -
 * Folder ကို **နာမည်အဆန်း** ပေးထားပါ။ (ဥပမာ - "Project Old Files" သို့မဟုတ် "Driver Backup")
 * ဖြစ်နိုင်ရင် Folder ကို **Password** ခံထားတာမျိုး ဒါမှမဟုတ် မျက်စိထဲ မမြင်သာတဲ့ နေရာမှာ ဝှက်ထားတာမျိုး လုပ်သင့်ပါတယ်။
ဒါပေမဲ့ အကောင်းဆုံးကတော့ စိတ်အေးရအောင် **လုံးဝ ဖျက်ပစ်လိုက်တာ** ပါပဲ။ အစ်ကို့ရဲ့ အနာဂတ် "နု" နဲ့ သာယာဖို့အတွက် အတိတ်က ပုံရိပ်တွေကို အနားမှာ မထားတာက ပိုစိတ်ချရပါတယ်။
Format ကိစ္စ အဆင်မပြေတာမျိုး ရှိရင် ပြန်ပြောပေးပါဦး၊ ကျွန်တော် ထပ်ကူညီပေးပါ့မယ်။
